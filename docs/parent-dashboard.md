# Parent Dashboard

The parent dashboard is the steward-facing control surface for review, safety, continuity, and homeschool-capable records.

The parent is the authority for accepting learning receipts in v1.

## Purpose

The dashboard helps the parent review learning events without turning the learner into a score object.

It should make learning visible, not over-controlled.

## Primary Areas

```text
Pending Reviews
Recent Receipts
Open Follow-Ups
Portfolio Map
Homeschool Export
Privacy Controls
AI Suggestion Review
```

## Pending Reviews

Shows sessions or draft receipts waiting for parent review.

The parent may accept, reject, edit, or mark for follow-up.

## Recent Receipts

Shows recently accepted receipts with activity type, learner explanation, parent note, and evidence items.

## Open Follow-Ups

Shows questions, projects, or receipts marked as needing another step.

Examples:

- ask again later
- repeat the experiment
- build a stronger version
- explain with a drawing
- compare two sources
- record a clearer explanation

## Portfolio Map

Shows longer-term learning continuity.

The map may show recurring interests, subject mappings, artifacts, and project clusters.

It must not turn patterns into fixed identity labels.

## Homeschool Export

Generates parent-reviewed records for a selected time period.

Exports may be grouped by:

- date
- subject
- project
- activity type
- evidence type
- receipt status

## Privacy Controls

Allows the parent to control storage, export, sharing, and public visibility.

Private by default is required.

## AI Suggestion Review

Shows AI-generated suggestions before they affect accepted records.

The parent may approve, edit, reject, or ignore AI suggestions.

## V1 Parent Actions

```text
accept receipt
reject receipt
request follow-up
edit parent note
add subject mapping
remove subject mapping
attach evidence
export records
mark private
```

## Dashboard Rule

The dashboard should support stewardship, not surveillance.

It should help the parent preserve evidence and protect the learner's growth without turning every moment into measurement.
